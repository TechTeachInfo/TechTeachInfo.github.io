Frontend

In this track you will learn to develop your own websites using the latest trends and technologies used in the field.

A frontend developer is the link between the graphical design, who design the website, and the backend developer, who will implement the logic required to handle the actions of the users of your website. These days the role of frontend developer is broad, creative an requires a lot of different skills because of all different devices your users will use these days to visit your website. Think of phones, tables and the ‘normal’ webbrowser.

In this track you will learn HTML, CSS and JavaScript. HTML is the main building block of website, using CSS you can give, your own unique, styling to your HTML and JavaScript is used to make your page interactive, like loading more content on the fly and give quite visual feedback to your users.

Into

This track will teach you HTML, CSS and JavaScript which are used to build websites. For every language you will first receive an introduction and after the into you can discover the languages more in depth.

Preparation

For this track you will need a text editor. We recommend you use Sublime Text; this text editor will give you some extra help while learning because it provides syntax (visual) coloring and some typing completions for HTML, CSS and Javascript. You can download Sublime Text here.

Starting point

This track has three seperate parts:

  • HTML
  • CSS
  • JavaScript

All parts have three different difficulty levels: beginner, intermediate and advanced. To start first do the 'beginner' level of every part. You can find the online course here. The course will explain every step so you can follow along.

Next steps
