Frontend web development

In this track you will learn to develop your own websites using the latest trends and technologies used in the field.

A frontend developer is the link between the graphical design, who designs the website, and the backend developer, who will implement the logic required to handle the actions of the users of your website. These days the role of frontend developer is broad, creative and requires a lot of different skills because of all different devices your users will use to visit your website. Think of phones, tables and the ‘normal’ webbrowser.

In this track you will learn HTML, CSS and JavaScript. HTML is the main building block of a website, using CSS you can give, your own unique, styling to your HTML and JavaScript is used to make your page interactive, like loading more content on the fly and give quite visual feedback to your users.

Intro

This track will teach you HTML, CSS and JavaScript which are used to build websites. For each subject you will first receive an introduction and after the intro you can explore them more in depth.

Preparation

For this track you will need a text editor. Microsoft Word is not very suitable to use for programming and other text editors will give you visual support, which comes in handy. We recommend you use Sublime Text; this text editor is for free and will give you some extra help while learning because it provides syntax coloring and some typing completions for HTML, CSS and Javascript. You can download Sublime Text here.
