Android

Smartphone apps these days are tremendously popular and a lot of companies have their own app for their customers or employees. That's why in this track you will learn how to build an app for Android phones.

As an Android app developer you will mainly use the programming language Java. Besides learning Java you will explore different parts of the Android Software Development Kit (Android SDK) to be able to use different features of a smartphone. For an app you will learn to build both the user interface (the part of the app that people see) and the logic that drives the app.

In this track you will learn the basics of Java and the Android SDK. We will use a real phone (or the simulator) to build and test the app.

Intro

In this track you will learn the basics of Java and the Android SDK. We will make use of a real Android phone to test the app. Tnstalling all software needed to be able to develop Android apps takes quite a lot of time, therefore we’ve created a virtual machine with an Android development environment pre-installed. This way you can start with the track immediately.

Enter the wonderful world of programming

Phone or Simulator

To follow the Android track it is recommended that you have a real Android phone. The phone needs to run at least Android 4.1, you can find the version of Android in the settings of your phone. Go to the Settings app and click on the item 'About this phone'. You will also need a USB cable to connect the phone to the laptop and test your app.
If you don't have an Android phone but still want to follow the track you can use the Android Simulator. The simulator is tasking for your system so it might not run fast enough for you to follow the track; please try to see if it works fast enough on your machine!

Android Studio

To get started with this track you should first install Anroid Studio; click on the link to download Android Studio.
When the download is finished start the installer and follow the steps outlined below:

  • Press 'next' on the first screen
  • If you have a real Android phone with Android 4.1 or highter deselect the option 'Android Virtual Device'; press 'next'
  • All other options can be left at the default settings; press 'next' until the installer starts to do its work
  • After installing you are on the final page; leave the option 'Start Android Studio' selected and press finish

Android Studio will start up for the first time and ask you to configure it; follow the steps outlined below to get it setup:
  • The first question will be if Android Studio has to import anything; select 'Do not import settings' and press 'OK'
  • On the intro screen press 'next'
  • On the next screen choose 'standard' and press 'next'
  • On this screen you can select your color preference; make your selection and press 'next'
  • Select 'finish', Android Studio will download all needed Android SDK components, this might take a while

When everything is downloaded Android Studio will show you it's main start page; you are now ready to follow the program!

Getting started with Android

Within the provided virtual machine there is an IDE (Integrated Development Environment) installed named ‘Android Studio’. Android Studio is the program you use to develop and test Android apps. Also we don’t make use of the Android Virtual Device (AVD) simulator but instead we test on a real device, so at figure 9 your real device should show up when you press the ‘play’ button. You can skip figure 10 to 13 because they describe how to setup the AVD.
