Android

Smartphone apps these days are tremendously popular and a lot of companies have their own app for their customers or employees. That's why in this track you will learn how to build an app for Android phones.

As an Android app developer you will mainly use the programming language Java. Besides learning Java you will explore different parts of the Android Software Development Kit (Android SDK) to be able to use different features of a smartphone. For an app you will learn to build both the user interface (the part of the app that people see) and the logic that will drive your app.

In this track you will learn the basics of Java and the Android SDK. We will use a real phone to build and test our app.

Intro

In this track you will learn the basics of Java and the Android SDK. We will make use of a real Android phone to test and debug the app. Because installing all software needed to be able to develop Android apps takes quite a lot of time we’ve created a ‘virtual machine’ with a Android development environment pre-installed. This way you can start with the track immediately.

Enter the wonderful world of programming

Phone

To follow the Android track you will need a real Android phone. The phone needs to run at least Android 4.1, you can find the version of Android in the settings of your phone (usally the settings app and then the item 'About this phone'). You will also need a USB cable to connect the phone to the laptop and test your app.

VirtualBox

To get started with this track you should first install VirtualBox and download the Virtual Machine for this track, here: [link]. When you’ve installed Virtaulbox you can import the machine. To do so open VirtualBox go to File and click on Import Appliance. In the popup navigate to the folder in which you’ve downloaded the ova file and open it. Import the machine by pressing ‘Start’.

Getting started with Android

Within the provided virtual machine there is an IDE (Integrated Development Environment) installed named ‘Android Studio’. Android Studio is the program you use to develop and test Android apps. We will be following an online course containing talks and practical labs. Go to this website: link. Start by reading the talks:

  • App Design
  • Android Overview
  • Android Anatomy
You don’t have to watch the videos.

Then get started on Lab-01. Warning: when starting with Lab-01 you can skip the first step (the steps are visible in the right top hand corner of the screen), so start at step ’02’. The installation of the software (that they describe in step ’01’) has already been done in the virtual machine. Also we don’t make use of the Android Virtual Device (AVD) simulator but instead we test on a real device, so at figure 9 your real device should show up when you press the ‘play’ button. You can skip figure 10 to 13 because they describe how to setup the AVD.
