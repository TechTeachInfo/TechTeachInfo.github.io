Backend web development

In this track you will learn to develop the logic that goes on behind the screens of a website. For example, when booking flight tickets online; the user will press the button ‘book’ and the backend will be responsible for actually making the booking. This will probably consist of storing the information, making a reservation with the airline software systems and sending email confirmations.

A backend developer creates all the logic to process incoming user data by validating it, storing the data in a structure and eventually loading the information again to present it back to the user. And usually a backend developer also connects a lot with other external systems to send emails or notifications for example.

In this track you will learn the basis of the programming language Java. Next, you will develop a web backend who will respond to user requests from the web. The backend will read and write data into a storage system, a so-called database, the specific database you will be using is called PostgreSQL.

Databases

Backend systems store information; a lot of times this information is stored in a database. There are a lot of different databases but the most common type of database is a relational database. Data is stored in tables and there exist relations between the tables to link them to each other. For example, when you are creating a webshop backend there will be a table for products and one for orders. A relation must be defined between those two tables since an order contains one or more products.

Most of the relational databases support a query language called SQL to extract information from them. In this part of the track you will learn the basics of SQL.

To learn SQL you will follow an online course. This makes it easier to get started because you don't need to install and configure the database sofware yourself. Follow these steps to get started:

  • Go to Codecademy and create an account.
  • Next, go to the SQL course

You can work though the course in your browser. Every section of the course contains some explanation and background information and you will be able to test your skills in exercises.
